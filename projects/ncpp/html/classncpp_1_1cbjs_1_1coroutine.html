Detailed Description

A coroutine is a function that can suspend execution to be resumed later.

It contain a fiber to run the main worker loop, be able to switch back to the owner thread fiber and resumed later by switching to that fiber.

Changes the state updater to default and switch to the owner thread fiber.

Also resetting job_p_ to NULL and state_ to coroutine_state::RUNNABLE. Be private because this function only need to call by the coroutine its self to switch back to the owner thread fiber when the coroutine is ready to take a new job.
